# CLAUDE.md

本文件为 Claude Code (claude.ai/code) 在本仓库工作时提供指引。

## 项目概览

这是一个个人技术博客仓库（build-blog），文章按年份归档。内容以中文技术文章为主，涵盖前端、Node.js、Linux 与 JavaScript 等主题。

## 仓库结构

- **按年份组织**：文章按年份目录存放（例如 2017/、2018/、2025/）
- **命名规则**：文章采用 `00X_标题.md` 格式，X 为递增编号
- **内容范围**：中文技术文章，主要包括：
  - 前端开发
  - Node.js 应用与部署
  - JavaScript 基础与进阶
  - Linux 服务器与运维
  - 开发工具与工作流

## 文件管理

- **语言**：默认中文内容
- **格式**：Markdown，包含技术说明、代码示例与教程结构
- **资源**：静态资源存放在 `assets/` 目录
- **版本控制**：Git 跟踪所有文章与索引更新

## 内容规范

在本仓库工作时：
- 保持按年份的目录结构
- 新文章沿用编号命名规则
- 除非明确要求，否则内容使用中文
- 保留技术教程风格（示例 + 解释）
- 新增文章后同步更新 `README.md` 索引

## 开发说明

- 这是一个静态内容仓库，无构建流程或依赖
- `index.js` 目前为空
- 无 `package.json` 或构建工具
- 内容通过直接编辑 Markdown 管理
